# AI Instructions

When generating code for this project follow these rules:

1. FleetCore is the source of truth.
2. Fleet tasks must not be permanently stored.
3. Manual tasks are stored in the tasks table.
4. The My Day screen must combine fleet alerts and manual tasks.
5. Completing a fleet task must update the related fleet table.
6. Attachments must support images and documents.
7. The UI must prioritize critical alerts first.
