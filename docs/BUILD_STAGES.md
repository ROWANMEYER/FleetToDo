# Build Stages

## Stage 1 — Basic Planner

Create manual tasks.

Features:

* create task
* mark complete
* priority levels
* My Day screen

## Stage 2 — Fleet Integration

Connect planner to fleet tables.

Generate tasks from:

* driver PrDP expiry
* truck license expiry
* trailer license expiry

## Stage 3 — Attachments

Allow users to attach files to tasks.

Supported types:

* images
* pdf documents

## Stage 4 — Fleet Operations Alerts

Add additional checks:

* missing documents
* overdue maintenance
* inspection reminders

## Stage 5 — Analytics

Provide reporting on completed tasks and compliance trends.
