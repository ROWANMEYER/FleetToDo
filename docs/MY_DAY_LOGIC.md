# My Day Logic

The My Day screen displays the most relevant tasks for the current day.

Tasks are generated from three sources.

1. Overdue Manual Tasks
2. Today's Manual Tasks
3. Fleet Alerts

Fleet alerts include:

* driver PrDP expiry
* driver license expiry
* truck license expiry
* trailer license expiry
* insurance expiry

Priority order:

Critical
Warning
Normal

Tasks are sorted by priority and due date.
